Title: Release 2.7 Date: Tue, 02 May 2017

The Wine Staging release 2.7 is now available.

What's new in this release:

  • Fixes for cabinet file handling in setupapi.
  • Cleanup of existing patches.
  • Various bug fixes.

In addition, Wine Staging users will also benefit from the following changes merged in the development branch:

  • TCP and UDP connection support in WebServices.
  • Various shader improvements for Direct3D 11.
  • Improved support for high DPI settings.
  • Partial reimplementation of the GLU library.
  • Window management improvements on macOS.
  • Various bug fixes.

The source is available now. Binary packages are in the process of being built, and will appear soon at their respective download locations.
